# Projeto Back-End

## Tecnologias utilizadas

- Java  17
- Spring boot 3.4
- Spring Security + JWT
- Spring Data JPA
- Flyway
- SGBD PostgreSQL
- OpenAPI (Swagger)
- Git/GitHub
- JavaMail (Notificação via e-mail)


## Algumas das configurações mais relevantes do projeto estão localizadas no aruqivo: application.properties dentro de src/main/resources
